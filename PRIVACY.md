# Privacy Policy

Last Updated: June 2026

This Discord bot collects and processes information necessary to provide its functionality within Discord servers.

## Data Collected

The bot may collect and process:

* Discord User IDs
* Discord Server IDs
* Channel IDs
* Role IDs
* Message IDs
* Command usage data
* Server configuration settings
* Moderation logs
* Member and presence information when required for bot features

## How Data Is Used

Data is used solely for:

* Providing bot commands and functionality
* Server management and moderation features
* Statistics and analytics
* Maintaining bot settings and configurations
* Preventing abuse and ensuring service reliability

## Data Sharing

The bot does not sell or share user data with third parties except when required by law.

## Data Retention

Data is retained only as long as necessary to provide bot functionality and may be deleted upon request where applicable.

## User Rights

Server administrators may request deletion of stored server data by contacting the bot owner.

## Contact

For questions regarding this Privacy Policy, contact the bot owner through Discord.
